# PickShot v3.0 사용 가이드

## 프로그램 소개

PickShot은 사진작가를 위한 **초고속 사진 선별(컬링) 도구**입니다.
JPG 파일을 이용한 빠른 미리보기로 수천 장의 사진을 신속하게 검토하고,
별점/셀렉으로 선별한 후 RAW 파일과 함께 내보낼 수 있습니다.

---

## 1. 시작하기

### 사진 불러오기
- 상단 **폴더 열기** 버튼 클릭 (또는 `Cmd + O`)
- JPG와 RAW가 같은 폴더에 있거나, 별도 폴더에 있어도 자동 매칭됩니다
- 지원 RAW: CR2, CR3, ARW, NEF, RAF, DNG, ORF, RW2

### 화면 구성
```
[상단 툴바: 폴더열기, 필터, 정렬, 내보내기, 분석 등]
[왼쪽: 썸네일 목록] | [오른쪽: 미리보기 + 메타데이터]
[하단: 보정, 별점, 줌 컨트롤]
```

## 3. 사진 선별 방법

### 별점 (★)
- 숫자키 `1~5`로 별점 부여
- 여러 장 선택 후 숫자키 → 일괄 별점
- 상단 필터에서 별점별 필터링 가능
- `0` 키로 별점 제거

### 스페이스 셀렉 (SP)
- `Space` 키로 빠르게 마킹
- 별점보다 빠른 "이것만 골라!" 용도
- 빨간 SP 아이콘으로 표시
- 라이트룸 내보내기 시 빨간 라벨로 변환

### G셀렉 (Google Drive 즉시 업로드)
- 상단 `G☁️` 버튼 → Google 로그인 → 폴더 생성
- `G` 키로 사진 선택 → Google Drive에 즉시 업로드
- 다시 `G` 키 → Google Drive에서 즉시 삭제
- 공유 링크로 클라이언트가 실시간 확인 가능
- QR 코드 생성하여 현장에서 클라이언트 폰으로 바로 스캔

## 4. 보기 모드

### 그리드 + 미리보기 (기본)
왼쪽 썸네일 + 오른쪽 미리보기. 가장 일반적인 작업 모드.

### 필름스트립
하단 가로 썸네일 + 상단 대형 미리보기. 사진 하나하나 집중할 때.

### 목록 보기
파일명, 촬영 정보가 한눈에 보이는 목록형.

하단 좌측 아이콘으로 전환 가능.

---

## 5. 미리보기 기능

### 확대/축소
- `Cmd + +` 확대, `Cmd + -` 축소
- 슬라이더로 자유 조절
- 맞춤 버튼으로 화면 맞춤
- 확대 시 우측 하단에 미니 네비게이터 표시

### 루뻬 (확대경)
- 미리보기에서 원하는 부분 클릭 → 250% 확대된 루뻬 표시
- 다시 클릭하면 루뻬 해제
- 초점 확인에 유용

### 히스토그램
- `H` 키로 표시/숨기기
- 우측 상단에 RGB 히스토그램 오버레이

---

## 6. 내보내기

### RAW 내보내기 (`Cmd + E`)
- 선별한 사진의 RAW 파일을 지정 폴더로 복사
- JPG는 제외하고 RAW만 내보냄

### Lightroom 내보내기
- RAW + XMP 사이드카 파일 생성
- 별점이 XMP에 자동 기록 → Lightroom에서 동일 별점 표시
- SP 선택 사진 → 빨간 라벨로 변환
- Lightroom 자동 실행 옵션

---

## 7. 품질 분석 (무료)

상단 **품질 분석** 버튼 클릭:
- 블러/초점 감지
- 노출 과다/부족
- 콘트라스트 부족
- 눈 감김 감지
- 흔들림 감지 (셔터속도 대비)
- 아웃포커싱/장노출 의도 자동 감지

썸네일에 등급 배지 표시: 우수 / 양호 / 보통 / 미흡 / 나쁨

---

## 8. 장면 분류 & 얼굴 그룹핑 (무료)

### 장면 분류
- 상단 **장면** 버튼 → 장면 분류 실행
- 인물, 풍경, 음식, 동물, 건물, 공연 등 자동 태깅
- 태그별 필터링 가능

### 얼굴 그룹핑
- 상단 **인물** 버튼 → 얼굴 그룹 실행
- 같은 인물이 포함된 사진을 자동 그룹핑
- 얼굴 썸네일로 누가 누군지 바로 확인
- 특정 인물만 필터링 가능

---

## 9. AI 기능 (Pro 구독)

### AI 스마트 분류
- 상단 **AI 분류** 버튼 → AI 스마트 분류 실행
- 클린샷/인물/그룹/군중/무대/분위기/디테일/비하인드/기념 자동 분류
- 활용도 판정: 즉시사용 / 편집후사용 / 참고용 / 삭제후보
- 0~100 활용도 점수

### AI 자동 보정
- 미리보기 하단 **AI 보정** 버튼
- 한국 화보 스타일 자동 보정
- 수평/수직 보정, 크롭, 색감 조정

### AI 사진 설명
- 우측 패널 **AI 분석** 섹션
- 사진 내용 자연어 설명, 보정 제안, 스타일 분석, 프로 평가

---

## 10. 파일 이름 변경

상단 **이름 변경** 버튼:
- 프리셋: 텍스트+번호, 날짜+텍스트+번호 등
- 구성 요소 순서 변경 (▲▼ 버튼)
- 적용 대상: 선택한 사진만 / 전체
- 미리보기에서 변경 전→후 확인
