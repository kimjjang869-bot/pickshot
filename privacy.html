개인정보 처리방침

최종 업데이트: 2026년 4월 16일

요약: PickShot은 사진 선별을 위한 macOS 네이티브 앱입니다. 사용자의 사진과 데이터는 모두 본인 기기 안에서만 처리되며, 저희 서버로 전송되거나 수집되지 않습니다.

1. 개요

PickShot("저희", "앱")은 사용자의 개인정보를 중요하게 생각합니다. 본 개인정보 처리방침은 PickShot macOS 앱이 어떤 정보를 수집하고 사용하는지 설명합니다.

2. 저희가 수집하지 않는 정보

PickShot은 다음 정보를 수집하지 않습니다:

  • 사용자의 사진, 이미지, RAW 파일
  • 파일 이름, 폴더 경로, 메타데이터(EXIF)
  • 얼굴 인식 데이터, 장면 분류 결과
  • 레이팅, 컬러 라벨, 선별 선택
  • 사용 통계, 분석 데이터, 크래시 리포트
  • IP 주소, 기기 식별자, 위치 정보
  • 개인 식별 정보(이름, 이메일, 전화번호 등)

모든 분석(품질 분석, 얼굴 감지, 장면 분류)은 사용자의 Mac 내부에서 Apple Vision 프레임워크를 사용해 처리됩니다. 인터넷 연결이 필요하지 않으며, 어떤 데이터도 외부로 전송되지 않습니다.

3. 선택적 기능과 제3자 서비스

3.1 Google Drive 연동 (선택)

"클라이언트 내보내기" 또는 "G셀렉" 기능을 사용하시면 PickShot이 Google Drive API에 접근할 수 있습니다. 이 경우 다음이 적용됩니다:

  • 사용자가 직접 선택하고 동의한 경우에만 작동합니다.
  • PickShot은 drive.file 범위만 요청합니다 — 앱이 생성한 파일만 접근할 수 있으며, 사용자의 다른 Google Drive 파일은 볼 수 없습니다.
  • OAuth 토큰은 macOS Keychain에 안전하게 저장되며, 저희 서버로 전송되지 않습니다.
  • 업로드된 파일은 사용자의 Google Drive에 저장되며, 저희는 이에 접근할 수 없습니다.
  • 언제든지 "로그아웃" 버튼으로 연동을 해제할 수 있습니다.

PickShot의 Google 사용자 데이터 사용은 Google API 서비스 사용자 데이터 정책(제한된 사용 요건 포함)을 준수합니다.

3.2 업데이트 확인

앱 실행 시 최신 버전이 있는지 확인하기 위해 GitHub Releases API(api.github.com)에 익명으로 요청을 보냅니다. 이 요청은 사용자를 식별할 수 있는 정보를 포함하지 않습니다.

3.3 App Store 결제

구독 및 결제는 Apple의 App Store 및 StoreKit을 통해 처리됩니다. PickShot은 사용자의 결제 정보(신용카드, 은행 정보 등)에 직접 접근하지 않습니다. Apple의 개인정보 처리방침을 참조하세요.

4. 로컬에 저장되는 데이터

다음 데이터는 사용자의 Mac에만 저장되며, 외부로 전송되지 않습니다:

  • UserDefaults: 앱 설정, 최근 폴더 목록, 즐겨찾기
  • Keychain: Google Drive OAuth 토큰(사용 시), 테스터 키(사용 시)
  • 캐시: 썸네일, 미리보기 이미지 (임시 디렉터리)
  • Security-Scoped Bookmarks: 사용자가 허용한 폴더 접근 권한

이 데이터는 앱 제거 시 또는 설정에서 캐시 삭제 시 제거됩니다.

5. 권한

PickShot은 macOS App Sandbox 환경에서 실행되며, 다음 권한을 요청합니다:

  • 사용자가 선택한 파일 읽기/쓰기: 사진 폴더 열기 및 편집
  • 네트워크 클라이언트: 업데이트 확인, Google Drive 연동(선택)
  • USB 장치: 카메라 테더링, 메모리 카드 접근
  • 사진 및 다운로드 폴더: 편의성을 위한 표준 접근
  • Keychain: 토큰 및 자격증명 안전 저장

6. 아동 개인정보 보호

PickShot은 만 13세 미만 아동을 대상으로 하지 않으며, 해당 연령의 아동으로부터 고의적으로 정보를 수집하지 않습니다.

7. 개인정보 처리방침 변경

본 방침이 변경될 경우 앱 업데이트 또는 본 페이지를 통해 공지하며, 중요한 변경사항은 별도로 안내합니다.
