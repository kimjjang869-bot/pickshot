## 1. 앱 소개

PickShot은 **세상에서 제일 빠른 뷰잉 + 쉬운 UI**를 목표로 만든 macOS 네이티브 사진 선별(Culling) 도구입니다.

촬영 후 수백~수천 장의 사진 중에서 좋은 사진을 빠르게 골라내고, 별점/셀렉으로 분류하고, Lightroom이나 폴더로 내보내는 전체 워크플로우를 하나의 앱에서 처리할 수 있습니다.

### 주요 특징

- **초고속 미리보기** -- Hardware JPEG 디코더 + Metal GPU 가속으로 RAW/JPG를 즉시 표시
- **JPG + RAW 자동 매칭** -- JPG로 빠르게 미리보기하고, 매칭되는 RAW 파일을 함께 관리
- **AI 분류** -- Vision 로컬 분류(무료) + Claude/Gemini AI 프롬프트 분류
- **NPU 보정** -- Apple Silicon NPU를 활용한 AI 보정, 디노이즈, 인물 인식 보정
- **메모리카드 백업** -- SD/CF 카드 연결 시 자동 감지 + 연속 백업
- **다양한 내보내기** -- 폴더, Lightroom(XMP), RAW to JPG 변환

### 지원 파일 형식

| 종류 | 확장자 |
|------|--------|
| JPG | .jpg, .jpeg |
| RAW | .cr2, .cr3, .nef, .arw, .orf, .raf, .dng, .rw2, .pef, .srw, .3fr |
| 기타 | .heic, .heif, .tiff, .tif, .png |

## 2. 시작하기

### 2.1 설치

1. DMG 파일을 다운로드합니다
2. DMG를 열고 PickShot 아이콘을 Applications 폴더로 드래그합니다
3. Applications에서 PickShot을 실행합니다

### 2.2 첫 실행

PickShot을 처음 실행하면 시작 화면이 나타납니다. 두 가지 방법으로 사진 폴더를 열 수 있습니다:

- **폴더 열기 버튼** 클릭 또는 `Cmd + O`
- 폴더를 앱 창에 **드래그 앤 드롭**

> JPG 폴더와 RAW 폴더가 분리되어 있어도 자동으로 매칭됩니다.

### 2.3 화면 구성

PickShot의 기본 화면은 세 영역으로 나뉩니다:

```
+----------------------------------------------+
|        툴바 (상단 2줄)                         |
+------------------+---------------------------+
|                  |                           |
|   폴더 브라우저   |     미리보기 영역          |
|   (왼쪽 사이드바) |                           |
|                  |                           |
+------------------+---------------------------+
|        썸네일 그리드 (하단)                    |
+----------------------------------------------+
```

- **툴바 1행**: 홈, 폴더 브라우저 토글, 경로, G셀렉, 매칭, 내보내기, 배치 처리
- **툴바 2행**: 정렬, 별점 필터, SP 필터, 검색, AI 분류, 레이아웃, 듀얼뷰, 비교, 지도, 슬라이드쇼, 테마, 도움말

## 3. 기본 사용법

### 3.1 썸네일 보기

#### 그리드 모드 (기본)
썸네일이 격자 형태로 표시됩니다. 각 썸네일에는 다음 정보가 표시됩니다:

- 파일 타입 배지 (JPG, R+J, RAW 등)
- SP(스페이스 셀렉) 표시
- G셀렉 표시
- 컬러 라벨
- 장면 태그
- 별점

#### 리스트 모드
파일 목록 형태로 표시되며, 이름/크기/수정일/종류별로 정렬할 수 있습니다.

#### 필름스트립 모드
미리보기를 크게 보면서 하단에 썸네일을 가로로 배치합니다. 툴바에서 레이아웃 버튼을 클릭하거나 `Cmd + Enter`로 전환합니다.

#### 썸네일 크기 조절
설정 > 미리보기 탭에서 기본 썸네일 크기를 50px ~ 300px로 조절할 수 있습니다.

### 3.2 미리보기

#### 확대/축소
| 동작 | 방법 |
|------|------|
| 확대 | `Cmd + =`, 마우스 스크롤 위, 핀치 제스처 |
| 축소 | `Cmd + -`, 마우스 스크롤 아래, 핀치 제스처 |
| 화면 맞춤 | `Cmd + 0`, 맞춤 버튼, 더블클릭(확대 상태에서) |
| 350% 확대 | 미리보기 영역 더블클릭 |
| 미들 마우스 클릭 | 화면 맞춤으로 돌아가기 |

확대 시 하단 오른쪽에 **미니 네비게이터**가 나타나며, 현재 보고 있는 영역을 표시합니다. 확대 상태에서 드래그하여 위치를 이동할 수 있습니다.

#### 줌 프리셋
화면 맞춤, 25%, 50%, 75%, 100%, 150%, 200%, 500%, 1000%, 2000%

#### 오버레이
- `H` -- 히스토그램 오버레이 토글
- `I` -- EXIF 메타데이터 오버레이 토글 (촬영 정보)

### 3.3 사진 탐색

#### 키보드
| 키 | 동작 |
|----|------|
| `<-` `->` | 이전/다음 사진 |
| `Up` `Down` | 위/아래 행 이동 |
| `Shift + 방향키` | 범위 선택 확장 |
| `Enter` | 선택된 폴더 열기 |
| `Cmd + Enter` | 필름스트립 전체화면 토글 |

#### 마우스
| 동작 | 방법 |
|------|------|
| 단일 선택 | 클릭 |
| 개별 추가/해제 | `Cmd + 클릭` |
| 범위 선택 | `Shift + 클릭` |
| 폴더 열기 | 폴더 썸네일 더블클릭 |

#### 전체 선택/해제
- `Cmd + A` -- 전체 선택
- `Cmd + D` -- 전체 해제
- 빈 영역 클릭 -- 전체 해제

### 3.4 정렬

툴바의 정렬 메뉴에서 선택:
- 촬영시간 (기본)
- 파일명 (오름차순/내림차순)
- 별점

### 3.5 검색

툴바의 검색창에 파일명을 입력하면 실시간으로 필터링됩니다. 검색 결과 수가 배지로 표시됩니다.

## 4. 사진 선별 (Culling)

### 4.1 별점 매기기

키보드 숫자키로 즉시 별점을 매길 수 있습니다:

| 키 | 동작 |
|----|------|
| `1` ~ `5` | 별점 1~5점 |
| `0` | 별점 초기화 |

여러 장을 선택한 상태에서 숫자키를 누르면 **선택된 모든 사진에 동시 적용**됩니다.

#### 별점 필터
툴바에서 All / 1 / 2 / 3 / 4 / 5 버튼을 클릭하면 해당 별점 이상의 사진만 표시됩니다.

### 4.2 SP 셀렉 (스페이스 셀렉)

**Space** 키를 누르면 현재 사진에 SP(스페이스 셀렉) 마크가 토글됩니다. 빠르게 좋은 사진을 골라낼 때 유용합니다.

- 썸네일 왼쪽 상단에 빨간 SP 배지가 표시됩니다
- 미리보기 하단에서도 SP 버튼으로 토글 가능합니다
- 툴바의 **SP** 필터 버튼으로 SP 사진만 필터링할 수 있습니다

### 4.3 컬러 라벨

사진에 색상 라벨을 붙여서 구분할 수 있습니다:

| 키 | 색상 |
|----|------|
| `6` | 주황 |
| `7` | 노랑 |
| `8` | 초록 |
| `9` | 파랑 |

우클릭 > 컬러 라벨 메뉴에서도 설정 가능합니다. "라벨 없음"으로 초기화할 수 있습니다.

### 4.4 비교 모드

2~4장의 사진을 선택한 후:
- 툴바의 비교 버튼 클릭 또는 `C` 키
- 선택한 사진들이 나란히 표시되어 비교할 수 있습니다

### 4.5 표정 비교

2~6장의 사진을 선택한 후:
- 툴바의 표정 비교 버튼 (얼굴 아이콘) 클릭
- 각 사진의 얼굴 부분을 확대하여 비교합니다

### 4.6 듀얼 뷰어

`D` 키 또는 툴바의 듀얼 뷰어 버튼을 눌러 두 장의 사진을 동시에 볼 수 있습니다.

### 4.7 전체화면

`Cmd + F` 또는 `Esc`로 전체화면을 토글합니다. `Cmd + Enter`로 필름스트립 전체화면으로 전환할 수 있습니다.

### 4.8 슬라이드쇼

툴바의 슬라이드쇼 버튼을 클릭하면 자동 재생 슬라이드쇼가 시작됩니다.

### 4.9 GPS 지도

GPS 정보가 있는 사진의 경우, 툴바의 지도 버튼을 클릭하면 촬영 위치를 지도에서 확인할 수 있습니다.

### 4.10 Quick Look

`P` 키를 누르면 macOS 기본 Quick Look으로 현재 사진을 미리봅니다.

### 4.11 실행취소

`Cmd + Z`로 마지막 별점/셀렉 변경을 실행취소할 수 있습니다.

## 5. AI 기능

### 5.1 Vision 로컬 분류 (무료)

Apple Vision 프레임워크를 사용하여 **인터넷 연결 없이** 사진을 자동 분류합니다.

**사용 방법:**
1. 툴바의 AI 메뉴 클릭
2. "Vision 로컬 분류" 선택

22개 이상의 카테고리로 분류되며, 분류 후 장면 태그로 필터링할 수 있습니다.

### 5.2 AI 프롬프트 분류 (Claude / Gemini)

Claude 또는 Gemini API를 사용하여 더 정교한 분류를 수행합니다. API 키가 필요합니다.

**프리셋 분류:**
툴바의 AI 메뉴 > AI 프롬프트 분류 섹션에서 미리 준비된 프리셋을 선택합니다.

**커스텀 프롬프트:**
"커스텀 프롬프트..."를 선택하여 직접 분류 기준을 입력할 수 있습니다.

**선택된 사진만 분류:**
전체가 아닌 선택된 사진만 분류할 수 있습니다.

**지원 엔진:**
| 엔진 | 특징 | 비용 |
|------|------|------|
| Gemini Flash | 최저가, 빠름 | $0.075/M 토큰 |
| Gemini Pro | 고성능 | $1.25/M 토큰 |
| Claude Haiku | 저렴, 빠름 | $0.25/M 토큰 |
| Claude Sonnet | 정확함 | $3/M 토큰 |

### 5.3 AI 분류 후 자동 폴더 정리

AI 분류 결과에 따라 사진을 자동으로 하위 폴더로 이동할 수 있습니다.

분류 결과 필터:
- 카테고리별: 클린샷, 인물, 그룹, 군중, 무대, 분위기, 디테일, 비하인드, 기념 등
- 활용도별: 즉시사용, 편집후사용, 참고용, 삭제후보

### 5.4 PickShot AI 분석

품질 분석 항목을 선택하여 실행합니다:
- **블러/초점 분석** -- 흔들림, 초점 나감 감지
- **눈 감김 감지** -- 인물 사진에서 눈 감은 얼굴 감지
- **인물 초점 분석** -- 얼굴 영역의 선명도 검사

분석 결과에 따라 AI 추천(AI Pick), 양호, 문제 있음으로 분류됩니다.

### 5.5 얼굴 그룹핑

인물 사진을 얼굴별로 자동 그룹핑합니다.

**사용 방법:**
1. 툴바의 인물 메뉴 클릭
2. "얼굴 그룹 실행" 선택
3. 완료 후 인물별로 필터링 가능

## 6. 보정

미리보기 하단 툴바에서 보정 기능에 접근합니다.

### 6.1 자동 보정

"보정" 메뉴 > "자동 보정"을 선택하면 보정 옵션 팝오버가 나타납니다.

Apple의 `autoAdjustmentFilters()`를 사용하여 자동으로 색보정, 노출 보정, 화이트밸런스를 조정합니다. Vision의 수평 감지를 활용한 수평/수직 보정도 포함됩니다.

여러 장을 선택한 상태에서 실행하면 **배치 보정**이 가능합니다.

### 6.2 NPU 보정 (Apple Silicon)

Apple Silicon의 Neural Processing Unit(NPU)를 활용한 고급 보정:

| 메뉴 | 기능 |
|------|------|
| AI 보정 (NPU) | 톤/색감 자동 최적화 |
| 디노이즈 | 고감도 노이즈 제거 |
| 인물 인식 보정 | 인물 영역과 배경을 분리하여 각각 최적 보정 |

### 6.3 AI 보정 (Pro)

Claude API를 사용하여 AI가 분석한 보정값을 적용합니다. API 키가 필요합니다. 보정 완료 후 적용 항목 요약과 API 비용이 표시됩니다.

### 6.4 가이드 보정 (Upright)

"보정" 메뉴 > "가이드 보정"을 선택하면 원근 보정 가이드가 나타납니다. 수직/수평선을 기준으로 원근 왜곡을 보정합니다.

### 6.5 크롭

**지원 비율:**
- 자유 크롭

- 3:2 / 2:3
- 4:3 / 3:4
- 16:9 / 9:16
- 스마트 크롭 (AI가 주요 피사체를 기준으로 자동 크롭)

### 6.6 무손실 JPEG 회전

미리보기 하단의 회전 버튼으로 90도 단위 회전이 가능합니다.

- 기타 파일: 픽셀 단위 회전

### 6.7 원본/보정 전환

보정 후 "원본/보정됨" 버튼으로 원본과 보정 결과를 즉시 비교할 수 있습니다. "저장" 버튼으로 보정된 이미지를 별도 파일로 저장합니다.

## 7. 내보내기

`Cmd + E` 또는 툴바의 "내보내기" 버튼을 클릭합니다.

### 7.1 내보내기 범위 선택

| 모드 | 설명 |
|------|------|
| 선택된 사진 | 현재 선택된 사진만 |
| 별점 있는 사진만 | 별점 1 이상인 사진 |
| 현재 필터 기준 | 현재 적용된 필터/검색 결과 |
| 전체 사진 | 모든 사진 |

### 7.2 폴더 내보내기

JPG와 RAW 파일을 지정한 폴더로 복사합니다.

- **JPG 폴더명 / RAW 폴더명** 커스터마이징 가능 (기본: JPG, RAW)
- 대상 폴더 선택 후 즉시 백그라운드에서 복사 시작
- 시트를 닫아도 백그라운드에서 계속 진행됩니다
- 완료 시 Finder에서 해당 폴더가 자동으로 열립니다

### 7.3 Lightroom 내보내기

RAW 파일 + XMP 사이드카 파일을 함께 내보냅니다.

- XMP에 별점 정보가 포함됩니다
- Lightroom에서 해당 폴더를 가져오면 별점이 자동 적용됩니다
- 내보내기 완료 시 Lightroom을 자동 실행할 수 있습니다 (설정에서 활성화)

### 7.4 RAW to JPG 변환

RAW 파일을 JPG로 일괄 변환합니다.

| 옵션 | 설명 |
|------|------|
| 해상도 | 원본, 4000px, 3000px, 2000px 등 |
| 품질 | 최고, 높음, 보통, 낮음 |
| 샤프닝 | 없음, 약간, 보통, 강하게 |
| 색공간 | sRGB, Display P3, Adobe RGB |
| 수평 보정 | 자동 수평 보정 적용 |
| 파일명 패턴 | 원본 이름, 접두사+번호 등 |

변환은 GPU 가속을 사용하며, 진행률/ETA가 표시됩니다. 변환 도중 중지 버튼으로 취소할 수 있습니다.

### 7.5 중복 처리

내보내기 시 대상 폴더에 이미 같은 파일이 있으면 중복 처리 다이얼로그가 나타납니다:

- **덮어쓰기** -- 기존 파일을 새 파일로 교체
- **이름 변경하여 내보내기** -- 새 파일에 번호를 붙여 저장
- **건너뛰기** -- 중복 파일은 복사하지 않음

### 7.6 배치 처리

툴바의 "배치 처리" 버튼으로 리사이즈 + 워터마크 일괄 처리가 가능합니다.

### 7.7 우클릭 내보내기

썸네일 우클릭 메뉴에서도 빠르게 내보내기할 수 있습니다:
- **내보내기** -- 내보내기 시트 열기
- **RAW to JPG 변환** -- 변환 모드로 바로 열기
- **Finder로 복사** -- 최근 5개 폴더 또는 새 폴더 선택

## 8. 폴더 관리

### 8.1 폴더 브라우저

왼쪽 사이드바의 폴더 브라우저에서 폴더 트리를 탐색합니다. 사이드바 토글: 툴바의 사이드바 버튼 클릭.

표시되는 위치:
- Desktop (자동 펼침)
- Documents
- Downloads
- Pictures
- 외장 디스크 (/Volumes)

폴더를 클릭하면 해당 폴더의 사진이 로드되고, 하위 폴더가 있으면 자동으로 트리가 펼쳐집니다.

### 8.2 하위 폴더 포함 열기

폴더를 우클릭 > "하위 폴더 포함 열기"를 선택하면 해당 폴더와 모든 하위 폴더의 사진을 한꺼번에 표시합니다. 상단에 주황색 "하위 폴더 포함" 배지가 나타나며, 클릭하면 현재 폴더만 표시 모드로 돌아갑니다.

### 8.3 새 폴더 만들기

다음 위치에서 새 폴더를 만들 수 있습니다:
- 폴더 트리에서 우클릭 > "새 폴더 만들기"
- 즐겨찾기에서 ... 메뉴 > "새 폴더 만들기"
- 썸네일 영역 우클릭 > "새 폴더 만들기"

### 8.4 폴더 이름 변경

폴더 트리에서 우클릭 > "폴더 이름 변경"을 선택합니다. 인라인 편집으로 바로 이름을 변경할 수 있습니다.

### 8.5 파일 드래그 이동

썸네일을 드래그하여 폴더 트리의 다른 폴더로 이동할 수 있습니다.

- 여러 장 선택 후 드래그하면 선택된 모든 사진(JPG + RAW 쌍)이 함께 이동
- 썸네일 그리드에서 하위 폴더 썸네일 위로 드래그해도 이동 가능
- 상위 폴더(..) 위로 드래그하면 상위 폴더로 이동

### 8.6 즐겨찾기

자주 사용하는 폴더를 즐겨찾기에 등록합니다.

**추가:**
- 폴더 트리에서 우클릭 > "즐겨찾기에 추가"
- 폴더를 즐겨찾기 영역으로 드래그 앤 드롭

**관리:**
- 즐겨찾기 항목의 ... 메뉴:
  - **이름 변경** -- 표시 이름 변경 (실제 폴더 이름은 변경되지 않음)
  - **즐겨찾기에서 제거**
  - **Finder에서 열기**

각 즐겨찾기에는 해당 폴더의 첫 번째 이미지가 썸네일로 표시됩니다.

### 8.7 외장 디스크 추출

폴더 트리에서 외장 디스크 옆의 추출 버튼을 클릭하면 안전하게 디스크를 추출합니다. 해당 볼륨의 사진이 열려 있으면 자동으로 참조를 해제한 후 추출합니다.

## 9. 메모리카드 백업

### 9.1 자동 감지

SD/CF 메모리카드를 연결하면 DCIM 폴더를 자동 감지하여 백업 프롬프트가 나타납니다.

> 설정 > 일반 > "메모리카드 자동 백업"을 활성화해야 합니다.

### 9.2 백업 시작

1. 메모리카드 연결
2. 백업 프롬프트에서 대상 폴더 선택
3. 백업 자동 시작

### 9.3 진행률 표시

백업 중 다음 정보가 표시됩니다:
- 복사 진행률 (완료/전체)
- 전송 속도 (MB/s)
- 남은 시간 예상 (ETA)
- 총 용량

### 9.4 검증 및 재시도

- 각 파일 복사 후 **크기 비교 검증**을 수행합니다
- 실패 시 최대 **3회 자동 재시도**
- 이미 존재하는 파일은 크기가 동일하면 **자동 건너뛰기**

### 9.5 완료 후 자동 언마운트

백업이 성공적으로 완료되면:
1. `diskutil eject` 명령으로 메모리카드 자동 추출
2. "다음 카드 대기" 모드로 전환

### 9.6 다음 카드 자동 시작

첫 카드 백업 완료 후, 새 메모리카드를 연결하면 같은 대상 폴더로 **자동 백업이 시작**됩니다. 폴더를 다시 선택할 필요가 없습니다.

### 9.7 멀티 세션 동시 백업

여러 장의 메모리카드를 동시에 연결하면 각각 별도의 백업 세션으로 동시 진행됩니다.

## 12. FAQ / 문제 해결

### Q: RAW 파일이 표시되지 않아요
**A:** PickShot은 JPG 파일을 기준으로 미리보기하고, RAW 파일을 자동 매칭합니다. JPG가 없는 RAW만 있는 경우에도 내장 프리뷰를 추출하여 표시하지만, 일부 최신 RAW 형식(예: Nikon Z8/Z9 High Efficiency)은 시간이 더 걸릴 수 있습니다.

### Q: 사진 로딩이 느려요
**A:** 설정 > 성능 최적화에서 벤치마크를 실행하고, "속도 우선" 또는 "균형" 프로필을 적용해보세요. 미리보기 해상도를 낮추면 크게 빨라집니다.

### Q: AI 분류가 안 돼요
**A:** 설정 > AI 엔진에서 API 키를 입력하고 "테스트" 버튼으로 연결을 확인하세요. Vision 로컬 분류는 API 키 없이 무료로 사용 가능합니다.

### Q: 별점이 저장되지 않아요
**A:** 설정 > 일반 > "종료 시 별점/셀렉 자동 저장"이 활성화되어 있는지 확인하세요.

### Q: Lightroom에서 별점이 안 보여요
**A:** Lightroom 내보내기를 사용하면 XMP 사이드카 파일에 별점이 포함됩니다. Lightroom에서 해당 폴더를 "가져오기"할 때 "사이드카 XMP 적용" 옵션을 확인하세요.

### Q: 메모리카드를 연결했는데 백업 프롬프트가 안 나타나요
**A:** 설정 > 일반 > "메모리카드 자동 백업"이 활성화되어 있는지 확인하세요. DCIM 폴더가 있는 카드만 자동 감지됩니다.

### Q: G셀렉이 뭔가요?
**A:** G셀렉은 사진을 Google Drive에 즉시 업로드하는 기능입니다. 툴바의 "G" 버튼으로 세션을 시작한 후, `G` 키로 사진을 선택하면 자동 업로드됩니다. 공유 링크/QR 코드를 생성하여 클라이언트에게 즉시 전달할 수 있습니다.

### Q: 캐시가 너무 많은 디스크 공간을 차지해요
**A:** 설정 > 캐시에서 현재 캐시 크기를 확인하고, 캐시 삭제 버튼으로 정리할 수 있습니다. 썸네일 캐시 최대 크기를 줄이면 디스크 사용량을 제한할 수 있습니다.

### Q: 우클릭 메뉴에서 할 수 있는 것들이 뭔가요?
**A:** 썸네일 우클릭 메뉴:
- 별점 매기기 (0~5)
- SP/G 셀렉 토글
- 컬러 라벨 설정
- 내보내기 / RAW to JPG 변환
- Finder로 복사 (최근 폴더 목록 포함)
- 파일명 복사
- Finder에서 보기
- 연결 프로그램으로 열기 (Photoshop, Lightroom, Bridge, Capture One, 미리보기 등)
- 이름 변경
- 목록에서 제거 / 원본 삭제
- 새 폴더 만들기

### Q: 연결 프로그램으로 열기가 뭔가요?
**A:** 우클릭 > "연결 프로그램으로 열기"에서 현재 사진을 다른 앱으로 바로 열 수 있습니다. 설치된 앱을 자동 감지하며, Photoshop/Lightroom/Bridge/Capture One/미리보기가 바로 목록에 표시됩니다. "기타 프로그램 선택..."으로 다른 앱도 선택할 수 있습니다.
