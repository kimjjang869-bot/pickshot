## 1. 시작하기

### 앱 실행
PickShot을 실행하면 시작 화면이 나타납니다:
- **JPG + RAW 파일 연동하기** — JPG 폴더와 RAW 폴더를 매칭
- **뷰어** — 사진 폴더를 열어 선별 시작
- **테더링** — USB 카메라 실시간 촬영 (Coming Soon)

### 지원 파일 형식
| 구분 | 확장자 |
|---|---|
| **JPG** | .jpg, .jpeg |
| **RAW** | .cr2, .cr3 (Canon), .nef (Nikon), .arw (Sony), .raf (Fuji), .dng, .orf, .rw2, .pef 등 30+ |
| **이미지** | .heic, .heif, .psd, .tif, .tiff |
| **동영상** | .mov, .mp4, .avi, .m4v |

### JPG + RAW 자동 매칭
같은 파일명의 JPG와 RAW가 있으면 자동으로 매칭됩니다:
- `IMG_001.jpg` + `IMG_001.arw` → **R+J** (보라색 뱃지)
- JPG만 있으면 → **JPG** (파란색 뱃지)
- RAW만 있으면 → **ARW/CR3/NEF** (주황색 뱃지)

## 2. 화면 구성

### 상단 툴바 (Row 1)
`[홈] [폴더 토글] [경로] ··· [로딩바 000/000 남은시간] [매칭] [G☁] [내보내기] [Free]`

### 필터 바 (Row 2)
`[정렬] [★1~5] [SP] [품질▼] [장면▼] [인물▼] [AI분류▼] [라벨▼] [이름변경] [그리드] [비교] [지도] [슬라이드쇼] [설정]`

### 메인 영역
- **좌측** — 폴더 브라우저 (접기/펼치기 가능)
- **중앙** — 썸네일 그리드 (크기 조절 슬라이더)
- **우측** — 미리보기 + EXIF 정보 + AI 분석

## 3. 키보드 단축키

### 이동
| 키 | 동작 |
|---|---|
| ← → | 이전/다음 사진 |
| ↑ ↓ | 위/아래 행 이동 |
| Shift + 방향키 | 범위 선택 |
| Cmd + A | 전체 선택 |
| Cmd + D | 선택 해제 |

### 별점 & 셀렉
| 키 | 동작 |
|---|---|
| 1 ~ 5 | 별점 1~5 |
| 0 | 별점 제거 |
| Space | SP 셀렉 토글 |
| G | G셀렉 토글 |

### 보기
| 키 | 동작 |
|---|---|
| H | 히스토그램 표시/숨기기 |
| C | 비교 모드 (2~4장) |
| P | Quick Look 미리보기 |
| Cmd + F | 풀스크린 뷰어 |
| Cmd + / Cmd - | 확대 / 축소 |
| Cmd + 0 | 화면에 맞추기 |

### 파일
| 키 | 동작 |
|---|---|
| Cmd + O | 폴더 열기 |
| Cmd + E | 내보내기 |
| Cmd + C | 선택한 사진 파인더 복사 |
| Cmd + Z | 실행 취소 |
| Cmd + ? | 단축키 도움말 |

## 4. 사진 선별 & 별점

### 별점 시스템
- 숫자키 **1~5**로 별점 부여, **0**으로 제거
- 필터 바에서 별점별 필터링 가능
- 별점 있는 사진만 내보내기 가능

### SP 셀렉 (Space Pick)
- **Space** 키로 토글
- 고객이 선택한 사진 표시용
- 빨간색 **SP** 뱃지 표시

### 색상 라벨
- 우클릭 > 컬러 라벨에서 7가지 색상 선택
- 작업 흐름별 분류에 활용 (예: 빨강=긴급, 초록=완료)

### 다중 선택
- **Cmd + 클릭** — 개별 추가/제거
- **Shift + 클릭** — 범위 선택
- **Cmd + A** — 전체 선택

## 5. 미리보기 & 뷰잉

### 줌 & 팬
- **마우스 휠** — 줌 인/아웃
- **드래그** — 줌 상태에서 이미지 이동
- **줌 프리셋** — 맞춤 / 25% / 50% / 75% / 100% / 150%

### 이미지 회전
- 미리보기 하단 **↺ ↻** 버튼으로 90도 회전
- 실제 픽셀 회전 (프레임 크기 자동 조정)
- 사진 전환 시 자동 리셋

### 초점 맵
- 하단 **초점** 버튼으로 활성화
- 빨간색 = 선명한 영역, 투명 = 흐린 영역

### 히스토그램
- **H** 키 또는 하단 **|||** 버튼
- RGB 분포 실시간 표시

### 풀스크린 뷰어
- **Cmd + F**로 진입
- 방향키로 이동, 1~5 별점, Space SP 셀렉
- **Esc**로 나가기

## 6. AI 분석 & 분류

### 품질 분석
- 필터 바 > **품질** > "품질 분석 실행"
- 선명도(40%) + 노출(30%) + 구도(30%) - 이슈 감점
- 결과: A/B/C/D/F 등급
- **AI Pick**: 75점 이상 + 이슈 없음

### 장면 분류
- 필터 바 > **장면** > "장면 분류 실행"
- Apple Vision 프레임워크 (Neural Engine)
- 카테고리: 인물, 풍경, 음식/음료, 건물/건축, 도시/야경, 실내, 동물/식물 등
- 480px 저해상도로 빠른 분류

### 인물 그룹핑
- 필터 바 > **인물** > "인물 그룹핑 실행"
- 얼굴 감지 → 특징 추출 → 같은 사람 그룹화
- 640px + 병렬 처리로 고속
- 신뢰도 0.7 이상만 인식 (오탐 방지)

## 7. 매칭 셀렉

툴바 보라색 **매칭** 버튼 클릭:

### 1. 파일명 텍스트 매칭
- 고객이 보낸 파일명 목록을 붙여넣기
- 쉼표, 줄바꿈, 공백 구분 자동 인식
- 4단계 매칭: 정확 → 부분 → 숫자패턴 → 퍼지
- 매칭된 사진에 ★1 자동 부여

### 2. JPG 반환 매칭
- 고객이 반환한 JPG 폴더 선택
- 현재 RAW 폴더와 파일명 비교
- 정확/유사/번호 패턴 매칭

### 3. AI 사진 매칭
- 카톡으로 받은 사진 폴더 선택
- pHash (이미지 지문) + EXIF 촬영시간으로 매칭
- **API 비용 없음** — 100% 로컬 처리

## 8. G Select (고객 셀렉)

### 흐름
1. **G Select 시작** → Google Drive 로그인
2. 폴더명/업로드 타입 설정
3. **G** 키로 사진 셀렉 → 자동 업로드
4. **QR 코드** 생성 → 고객에게 공유
5. 고객이 **웹 뷰어**에서 사진 셀렉
6. `.pickshot` 파일이 Google Drive에 자동 저장

### 고객 웹 뷰어
- 모바일 퍼스트 (iPhone/Android)
- 탭해서 셀렉 → "셀렉 완료" 버튼
- 로그인 불필요

### QR 코드
- **고객 셀렉** 탭 — 웹 뷰어 링크 QR
- **Drive 폴더** 탭 — Google Drive 직접 링크 QR

## 9. 내보내기 & RAW 변환

**Cmd + E** 또는 툴바 **내보내기** 버튼:

### 폴더 내보내기
- JPG/RAW 분리 폴더로 복사
- 폴더명 커스텀 가능

### Lightroom 내보내기
- RAW + XMP 사이드카 (별점 포함)
- Lightroom에서 가져오면 별점 자동 적용

### RAW → JPG 변환
| 옵션 | 값 |
|---|---|
| **해상도** | 원본 / 4000px / 2000px / 1200px |
| **품질** | 최고 95% / 높음 90% / 보통 85% / 웹용 80% |
| **샤프닝** | 없음 / 약하게 / 보통 / 강하게 |
| **색공간** | sRGB / Display P3 / Adobe RGB |
| **자동 수평** | Vision 기반 수평 보정 |
| **파일명** | 원본 / 날짜_원본 / 접두사_번호 / 날짜시간_번호 |

- **GPU 가속** (CIRAWFilter + Metal)
- **진행률 표시** + 남은 시간
- **중지 버튼** (빨간 ■)
- **중복 파일 체크** (덮어쓰기/건너뛰기/취소)

### 우클릭 메뉴
- 선택한 사진에서 바로 **RAW → JPG 변환** 가능
- **Finder로 복사** (최근 폴더 표시)
- **파일명 복사**

## 10. 고급 기능

### 배치 이름 변경
- 패턴 조합: 텍스트 + 날짜 + 번호
- 프리셋 5종 제공
- 미리보기에서 결과 확인 후 실행

### 비교 모드
- **C** 키로 2~4장 나란히 비교
- 동기화 줌 & 팬
- EXIF 정보 + 별점 표시

### 슬라이드쇼
- 자동 재생 (간격 조절)
- 전환 효과: 페이드/슬라이드/줌/디졸브

### 지도 뷰
- GPS 촬영 위치 지도에 표시
- 클러스터 뷰 지원

### 로그 보내기
- PickShot 정보 > **로그 보내기**
- 3가지 방식: Google Drive / 이메일 / 파일 저장
- 성능 데이터만 전송 (개인정보 없음)

## 12. 문제 해결

### 앱이 느려질 때
1. 설정 > 성능 > 메모리 제한 확인
2. 썸네일 캐시 크기 줄이기
3. 프리뷰 해상도 제한 설정

### 썸네일이 안 나올 때
1. 파일 접근 권한 확인 (시스템 설정 > 개인정보 보호)
2. 외장 드라이브 연결 상태 확인
3. 지원되는 RAW 형식인지 확인

### 업데이트
- 앱 시작 시 자동 체크
- 새 버전 알림 > "다운로드" 클릭
- GitHub Releases에서 직접 다운로드 가능

### 로그 전송
- 문제 발생 시 **PickShot 정보 > 로그 보내기**
- 개발자에게 성능 데이터 전송으로 빠른 해결
